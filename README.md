# Campout

Real-time multiplayer browser-based survival game with full systems architecture
written from scratch.

## current state

**Phase 2 Complete** ✅

- Single-player playable game (Phase 0)
- Everything containerized with Docker Compose (Phase 1)
- Database and backend services running in containers (Phase 2)

You can run the full stack locally with `docker compose up --build` and visitng your localhost to play the game. Very barebones, but art and mechanics will be improved! Coming soon!
